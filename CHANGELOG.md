# История изменений

## 2.1.0 — 2026-07-20

- добавлен единый роутер команд пульта, клавиатуры и gamepad;
- навигация разделена на предсказуемые зоны без переходов через край;
- фокус запоминается для домашнего экрана, панелей и диалогов;
- исправлено подтверждение кнопок через OK/Enter;
- кнопки Назад и Home получили единое поведение во всей оболочке;
- Menu открывает панель управления активным веб-приложением;
- мультимедийные команды передаются последнему веб-приложению;
- аппаратная кнопка питания теперь открывает диалог подтверждения;
- сообщение сборщика автоматически использует текущую версию пакета.
