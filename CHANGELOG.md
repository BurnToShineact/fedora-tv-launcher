# История изменений

## 2.2.1 — 2026-07-20

- исправлен запуск системного окна подтверждения обновления в отдельной TV-сессии;
- добавлен и автоматически запускается графический агент PolicyKit;
- оболочка больше не сворачивается во время установки обновления;
- добавлен полноэкранный визуальный процесс с этапами подтверждения, установки и перезапуска;
- сообщения об отсутствии агента и отмене авторизации стали понятнее.

## 2.2.0 — 2026-07-20

- добавлены системные настройки громкости, отключения звука и Wi‑Fi;
- добавлено переключение русского и английского языка интерфейса;
- добавлена экранная клавиатура для полей оболочки и встроенных веб-приложений;
- встроенные приложения получили компактные фирменные иконки;
- добавленные системные приложения теперь используют иконки из Fedora;
- над внешними приложениями появилась плавающая кнопка возврата в Fedora TV;
- плитки приложений стали компактнее, чтобы на экране помещалось больше сервисов.

## 2.1.0 — 2026-07-20

- добавлен единый роутер команд пульта, клавиатуры и gamepad;
- навигация разделена на предсказуемые зоны без переходов через край;
- фокус запоминается для домашнего экрана, панелей и диалогов;
- исправлено подтверждение кнопок через OK/Enter;
- кнопки Назад и Home получили единое поведение во всей оболочке;
- Menu открывает панель управления активным веб-приложением;
- мультимедийные команды передаются последнему веб-приложению;
- аппаратная кнопка питания теперь открывает диалог подтверждения;
- сообщение сборщика автоматически использует текущую версию пакета.
